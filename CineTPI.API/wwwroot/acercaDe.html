Acerca del proyecto

CineTPI es una aplicación desarrollada como Trabajo Práctico Integrador para la materia Programación II (UTN Córdoba). El objetivo es simular la gestión de un cine, integrando:

  • Modelado de dominio y reglas de negocio con C# y POO avanzada.
  • Acceso a datos mediante una Web API.
  • Interfaz web en HTML, CSS y JavaScript.
  • Módulos de cartelera, funciones, reservas y panel de administración.
